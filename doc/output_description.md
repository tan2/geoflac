# GeoFLAC User Guide: Output Files & VTK Datasets

This guide documents the binary output files generated by the GeoFLAC solver, their physical meanings, units, and how they correspond to the datasets visible when visualizing the `.vts` and `.vtp` files in VisIt.

---

## 1. Solver Binary Output Files (`*.0`)

When a simulation is run, the solver writes outputs to binary files at regular time intervals (configured by `dtout-file` in the `.inp` file). The files are stored as a sequence of frames, with each frame containing float or integer values.

### A. General & Spatial Coordinate Files
*   **`time.0`**
    *   **Meaning**: The elapsed simulation time of each output record.
    *   **Units**: Seconds ($s$) in the raw file (often converted to thousands/millions of years, $Kyrs$ or $Myrs$, in post-processing).
*   **`mesh.0`**
    *   **Meaning**: Grid node coordinate arrays (containing $x$-coordinates followed by $z$-coordinates for all $nx \times nz$ nodes).
    *   **Units**: Kilometers ($\text{km}$).

### B. Node-Based Field Outputs
*   **`vel.0`**
    *   **Meaning**: Horizontal ($V_x$) and vertical ($V_z$) velocities at grid nodes.
    *   **Units**: Centimeters per year ($\text{cm/yr}$).
*   **`temp.0`**
    *   **Meaning**: Temperature at grid nodes.
    *   **Units**: Celsius ($^\circ\text{C}$).

### C. Element-Based Field Outputs
*   **`phase.0`**
    *   **Meaning**: Rock lithological phase ID of each grid cell.
    *   **Units**: Dimensionless (Integer phase IDs, $1 \le \text{ID} \le N$).
*   **`visc.0`**
    *   **Meaning**: Effective shear viscosity of the grid cell.
    *   **Units**: $\log_{10}(\text{Pa}\cdot\text{s})$ (e.g., a value of $22.0$ represents $10^{22} \text{ Pa}\cdot\text{s}$).
*   **`aps.0`**
    *   **Meaning**: Accumulated plastic strain (total non-elastic shear deformation accommodated by the cell).
    *   **Units**: Dimensionless (fraction).
*   **`pres.0`**
    *   **Meaning**: Hydrostatic confining pressure.
    *   **Units**: Kilobars ($\text{kbar}$, where $1 \text{ kbar} = 10^8 \text{ Pa} = 100 \text{ MPa}$).
*   **`srII.0`**
    *   **Meaning**: Second invariant of the deviatoric strain rate tensor.
    *   **Units**: $\log_{10}(\text{s}^{-1})$ (e.g., a value of $-14.0$ represents a strain rate of $10^{-14} \text{ s}^{-1}$).
*   **`eII.0` & `eI.0`**
    *   **Meaning**: Second invariant of deviatoric strain, and first invariant of strain (volumetric strain).
    *   **Units**: Dimensionless (fraction).
*   **`sII.0`**
    *   **Meaning**: Second invariant of deviatoric stress (effective stress).
    *   **Units**: Kilobars ($\text{kbar}$).
*   **`sxx.0`, `szz.0`, `sxz.0`**
    *   **Meaning**: Deviatoric stress tensor components ($\sigma'_{xx}$, $\sigma'_{zz}$, and shear stress $\sigma_{xz}$).
    *   **Units**: Kilobars ($\text{kbar}$).
*   **`density.0`**
    *   **Meaning**: Effective bulk density of the element (including thermal expansion and pressure compressibility).
    *   **Units**: $\text{kg/m}^3$.
*   **`src.0`**
    *   **Meaning**: Heat source term (includes radiogenic decay and shear heating).
    *   **Units**: $\text{W/kg}$ (or $\text{W/m}^3$).
*   **`diss.0`**
    *   **Meaning**: Energy dissipation rate due to plastic and viscous work.
    *   **Units**: $\text{W/kg}$ (or $\text{W/m}^3$).
*   **`coolingrate.0`**
    *   **Meaning**: Cooling rate ($dT/dt$) tracked for markers.
    *   **Units**: $^\circ\text{C}/\text{Myr}$ (or $^\circ\text{C}/\text{s}$).

### D. Boundary & 1D Surface Outputs
*   **`forc.0`**
    *   **Meaning**: Integrated horizontal forces acting on the left and right side boundaries.
    *   **Units**: Newtons per meter ($\text{N/m}$, force per unit thickness).
*   **`hfl.0`**
    *   **Meaning**: Surface heat flow profile.
    *   **Units**: $\text{mW/m}^2$.
*   **`topo.0`**
    *   **Meaning**: Surface topography profile.
    *   **Units**: Kilometers ($\text{km}$).

## 2. VTK Dataset Reference (for VisIt)

When VisIt loads `.vts` (grid) or `.vtp` (marker) files converted via the utilities in `util/`, it presents the variables with standardized VTK dataset names. Below is the mapping between the VisIt datasets, their source output files, and their units.

### A. Grid Data (`.vts`)

| VisIt Dataset Name | Source Binary File | Dimension / Type | Units | Physical Meaning |
|:-------------------|:-------------------|:-----------------|:------|:-----------------|
| **`Velocity`** | `vel.0` | 3D Vector | $\text{m/s}$ | Velocity vector field ($V_x$, $V_z$, with $V_y=0.0$). Converted from $\text{cm/yr}$ during VTK export. |
| **`Temperature`** | `temp.0` | Scalar | $^\circ\text{C}$ | Temperature. |
| **`Cooling rate`** | `coolingrate.0` | Scalar | $^\circ\text{C}/\text{s}$ | Rock cooling rate ($dT/dt$). |
| **`Phase`** | `phase.0` | Scalar (Int) | Dimensionless | Rock type phase ID. |
| **`Plastic strain`** | `aps.0` | Scalar | Dimensionless | Accumulated plastic strain. |
| **`Strain rate`** | `srII.0` | Scalar | $\log_{10}(\text{s}^{-1})$ | Second invariant of deviatoric strain rate. |
| **`eII`** | `eII.0` | Scalar | Dimensionless | Second invariant of deviatoric strain. |
| **`Stress`** | `sII.0` | Scalar | $\text{kbar}$ | Second invariant of deviatoric stress. |
| **`Pressure`** | `pres.0` | Scalar | $\text{kbar}$ | Hydrostatic confining pressure. |
| **`Sxx`, `Szz`, `Sxz`**| `sxx.0`, `szz.0`, `sxz.0`| Scalar | $\text{kbar}$ | Deviatoric stress tensor components. |
| **`Density`** | `density.0` | Scalar | $\text{kg/m}^3$ | Bulk density. |
| **`Viscosity`** | `visc.0` | Scalar | $\log_{10}(\text{Pa}\cdot\text{s})$ | Effective shear viscosity. |
| **`Dissipation`** | `diss.0` | Scalar | $\text{W/kg}$ | Mechanical energy dissipation rate. |
| **`Melt fraction`** | `melt.0` | Scalar | Dimensionless | Partial melt volume fraction. |
| **`Magma fraction`**| `magma.0` | Scalar | Dimensionless | Migrating magma volume fraction. |
| **`Area`** | Calculated | Scalar | $\text{m}^2$ | Area of the grid element. |

### B. Lagrangian Marker Data (`.vtp`)

| VisIt Dataset Name | Dimension / Type | Units | Physical Meaning |
|:-------------------|:-----------------|:------|:-----------------|
| **`Phase`** | Scalar (Int) | Dimensionless | Phase ID of the marker. |
| **`age_ZFT`** | Scalar | $\text{Myr}$ | Zircon Fission Track cooling age. |
| **`age_ZHe`** | Scalar | $\text{Myr}$ | Zircon (U-Th)/He cooling age. |
| **`age_AFT`** | Scalar | $\text{Myr}$ | Apatite Fission Track cooling age. |
| **`age_AHe`** | Scalar | $\text{Myr}$ | Apatite (U-Th)/He cooling age. |
